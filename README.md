# abstract

### audiovisual entertainment

## usage
install
```sh
yarn install && yarn upgrade-interactive --latest
```

build
```sh
yarn build
```

run
```sh
yarn start
```

clean-up (delete build and release dirs: [rel, dist])
```sh
yarn clean
```

remove node_modules and clean-up (delete dirs: [node_modules, rel, dist])
```sh
yarn remove
```
